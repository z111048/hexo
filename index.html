CSS基礎設計

第二天：CSS 基礎設計

今天開始學習 CSS，用來美化和設計 HTML 網頁。CSS 讓我們可以改變網頁的外觀和佈局，我學會了如何使用選擇器來選擇元素，並且應用樣式規則。了解了常見的屬性如顏色、字體、邊距和填充，以及如何使用盒模型來控制元素的大小和邊框。這一天的學習讓我能夠初步設計出簡單而美觀的網頁，未來將繼續學習更進階的 CSS 技巧。

HTML基礎學習

第一天：HTML 基礎學習

今天專注於學習 HTML 的基本概念和標籤。HTML 是建立網頁的基礎語言，我從最簡單的結構開始，了解了如何使用標籤來定義標題、段落、連結和圖片。學會了如何用列表和表格來組織資訊，並且初步了解了表單的使用方法。這一天的學習讓我對 HTML 有了清晰的概念，期待明天深入學習 CSS，讓網頁更具吸引力。
